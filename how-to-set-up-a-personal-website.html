How To Set Up A Personal Website At The University Of Michigan

This is a basic how to make your own web page and hosting it on UM servers using my experience making this website (this is just one way to do it - using the terminal). I had some trouble understanding the instructions so hopefully this will help.

There are at least three UM departments that host websites:

  • IT
  • EECS
  • School of Information

This tutorial is for making a site on the general UM server(IT)

  1. Open the terminal and type in "ssh sftp.itd.umich.edu" always without quotes for every step. Also, if you have a CAEN account, "ssh uniquename@login.engin.umich.edu" works too.
  2. Go to the folder where the file is stored. Type in "cd Public/html". Type in "ls" to check what files are in this folder, currently it should only contain index.html.
  3. To open the file, one way is to use vim, it is easy to get trapped if you are a first time user (careful!). Type in "vim index.html" to enter the file and edit it.
  4. To start editing, type in "i" and delete all the default text. Write in the file using HTML syntax.
  5. To stop editing, press the esc button, then type ":", "w", and "q" to leave the file.
  6. To publish the site, type "chmod -R a+rX ~/Public/html".
  7. Check your new site at http://www.umich.edu/~uniqname/
